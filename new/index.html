Frequently Asked Questions

Find answers to common questions about Smart Scroll

What devices work with Smart Scroll?

+

Smart Scroll works with any device that has media volume controls, including keyboards with volume knobs, media keyboards, and external volume controllers recognized by Windows as media keys.

Will Smart Scroll affect my normal volume controls?

+

No, Smart Scroll only intercepts volume controls when in an active scrolling mode. Your volume controls work normally at all other times. You can toggle scrolling mode on or off anytime.

How do I switch between modes?

+

You can switch modes using the WIN+Shift+F1 hotkey or through the system tray icon. Three modes are available: Mute Mode (activate scrolling when muted), Scroll Lock Mode (when Scroll Lock is ON), and Num Lock Mode (when Num Lock is OFF).

Does Smart Scroll work in all applications?

+

Smart Scroll works in most Windows applications that support standard scrolling. However, some applications with custom interfaces or scrolling behaviors may have limited compatibility.

Is Smart Scroll open source?

+

Yes, Smart Scroll is an open-source project. You can view the source code and contribute on GitHub.
